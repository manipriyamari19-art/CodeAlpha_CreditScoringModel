# CodeAlpha_CreditScoringModel
Machine Learning Credit Scoring Model using Random Forest
# Credit Scoring Model

## Objective
Predict an individual's creditworthiness using financial data.

## Algorithm
Random Forest Classifier

## Features
- Income
- Loan Amount
- Credit History
- Employment Length

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## dataset used
- credit risk dataset
